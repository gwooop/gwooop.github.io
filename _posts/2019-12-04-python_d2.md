7. 임의 개수의 인수를 전달받아 가장 큰 숫자를 찾아 리턴하는 함수를 작성하라.

def maxnum(*args):
    biggest = args[0]
    for num in args:
        if num > biggest:
            biggest = num
    return biggest


print("가장 큰 수는?", maxnum(7,3,78,1,5))
